# discid

Command-line tool to extract MusicBrainz and FreeDB disc ids from audio CDs using [libdiscid](http://github.com/metabrainz/libdiscid).

## Prerequisites

Installed Ubuntu packages:
- libdiscid
- libdiscid-dev
